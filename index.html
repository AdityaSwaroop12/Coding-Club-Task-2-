How to Play Sudoku

  • The goal of Sudoku is to fill a 9x9 grid with numbers so that each row, column, and 3x3 section contains the digits 1 through 9 without repeating.
  • Each puzzle starts with some cells pre-filled. Your task is to complete the grid following the rules
  • Click a cell to select it, then choose a number from the number pad.
  • You can erase a number by selecting the cell and clicking the delete(‚ùå) button.
  • Use the timer to track your performance.
  • If you're stuck, you can start a new game or pause and Resume to take a break.
